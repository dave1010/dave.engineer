This site is a static build assembled with Eleventy and styled with Tailwind CSS. Markdown is used for long-form content, with Nunjucks templates supplying layouts and reusable components.

### Fonts

- **IBM Plex Mono** for mono accents.
- **Space Grotesk** for headings and body text.

### Tooling

- **PostCSS + Tailwind CSS** for production styles.
- **@11ty/eleventy-plugin-syntaxhighlight** (Prism) for code blocks.
- **Biome** for linting and formatting.

### Data & automation

- Content collections are curated in Eleventy.
- Deploys are kept tidy with automated checks and tests.
